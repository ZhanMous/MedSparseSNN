## 1. 核心类比体系

| 专业术语 | 非专业类比 | 固定视觉符号 |
|----------|------------|--------------|
| 传统ANN AI模型 | **「死记硬背的实习医生」**：背下了所有训练过的细胞图片，会泄露患者病历，看片必须整张图全看，费眼睛费时间 | 🧠🔓（灰色的大脑+撬开的锁+红色感叹号 |
| SNN/MedSparseSNN模型 | **「经验丰富的专科医生」**：只看细胞的关键特征，不会死记硬背患者病历，看片只看重点，又快又省力 | 🧠🔒（蓝色的大脑+闭合的锁+绿色对勾 |
| 稀疏性 | 医生只看细胞的核型、颗粒度等关键诊断特征，不盯着空白/无关像素 | 🔍✨（放大镜只聚焦核心区域+其余虚化） |
| 成员推理攻击(MIA) | 黑客想通过AI模型反推出「这个患者的片子有没有被用来训练AI」，窃取患者隐私 | 🎭📋（戴面罩的黑客小人+试图偷看病历本） |
| 低功耗推理 | 便携血液检测设备用这个AI，不会卡顿、不会耗电快，不用频繁充电 | 🔋⚡（绿色满格电池标识） |
| 随机猜测水平(0.500) | 黑客想偷隐私，结果跟抛硬币猜正反面一样，什么都偷不到 | 🪙（抛硬币图标） |

## 3. 设计规范强制约束

### 配色规范
- 主色调：医疗蓝+安全绿
- 警示色：浅红
- 全程不超过3种主色，避免花里胡哨

### 字体规范
- 标题用粗体无衬线字体
- 正文用常规无衬线字体
- 字号不小于24号（投影能看清）

### 内容约束
- 每页文字不超过3行
- 核心数据用超大字号突出
- 拒绝大段文字

### 图片规范
- 所有配图用高清实拍图/极简简笔画
- 拒绝模糊的学术截图、复杂的折线图

### 类比统一
- 全程贯穿「医生看片」的核心类比
- 不随意更换类比体系
